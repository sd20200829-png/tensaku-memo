# 添削メモツール

Markdownファイルを左右2画面で開き、添削メモを付けてAI向けに出力するブラウザベースの軽量ツール。

## 特徴

- **依存なし** — 単一HTMLファイル、ブラウザだけで動作
- **2画面構成** — 左：原文（読み取り専用）、右：添削メモ（インライン編集）
- **AI向け出力** — 行番号・原文引用・メモ・修正案を構造化Markdownで出力
- **ラウンドトリップ** — 出力ファイルを再読込するとメモが復元される

## クイックスタート

### ブラウザで直接開く

1. `tensaku-memo.html` をブラウザで開く
2. 「ファイルを開く」またはドラッグ＆ドロップでMDファイルを読み込む
3. 左パネルでテキストを選択 → 「メモ追加」→ 右パネルでメモを記入
4. 「エクスポート」で保存

### 右クリックメニューから開く（Windows）

1. `install-menu.cmd` をダブルクリック → 右クリックメニューに登録
2. MDファイルを右クリック →「添削メモで開く」（Shift+右クリックでクラシックメニュー）
3. Edge で右寄せウィンドウが開き、ファイルが自動読込される

## 使い方

### メモの追加

| 操作 | 動作 |
|------|------|
| 行をクリック | その行にメモを追加 |
| テキストを選択 →「メモ追加」 | 選択範囲にマーカー付きメモを追加 |
| 左のマーカーをクリック | 右の対応メモにジャンプ |
| メモカードのヘッダをクリック | 左の対応行にスクロール |

### メモの編集

- メモ欄と修正案欄に直接入力（修正案は任意）
- 両方空のまま別の場所をクリックすると自動削除
- 同じ行への重複メモは作成されない（既存メモにフォーカス）

### キーボードショートカット

| キー | 動作 |
|------|------|
| `Ctrl+E` | エクスポート |
| `Escape` | 選択解除 |

## 出力形式

`添削メモ_{元ファイル名}.md` として出力。元の本文の末尾に添削メモが付与される。

```markdown
（原文テキスト）

---

<!-- TENSAKU_MEMO_START -->
<!-- TENSAKU_DATA
[JSON形式のメモデータ]
TENSAKU_DATA -->

# 添削メモ (2件)

## L5 「静かに」
> 彼女は静かに部屋を出ていった。

- **メモ**: 冗長な表現
- **修正案**: 彼女は部屋を出ていった。

## L12
> 指先が——痺れている。昨日より、はっきりと。

- **メモ**: ダッシュの使い方が効果的

<!-- TENSAKU_MEMO_END -->
```

- **JSONブロック**（HTMLコメント内）: ツールで再読込するとメモが完全復元される
- **Markdownブロック**: AIや人間がそのまま読める形式

## ファイル構成

```
tensaku-memo.html    # メインツール
open-tensaku.cmd     # 右クリックメニュー用ランチャー
open-tensaku.ps1     # MD読込→一時HTML生成→Edge起動
install-menu.cmd     # 右クリックメニュー登録/削除
install-menu.ps1     # レジストリ操作
sample/              # サンプルファイル
```

## 右クリックメニューのアンインストール

`install-menu.cmd` を再度実行し、`y` を入力。

## 動作環境

- モダンブラウザ（Edge / Chrome 推奨）
- 右クリックメニュー機能: Windows 10/11 + PowerShell 5.1+
